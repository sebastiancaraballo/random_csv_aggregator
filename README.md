# random_csv_aggregator

## Installation
- Install `homebrew`
- Install `yarn`
- Install `ruby`
- Inside project directory run `bundle install`
- Run `rails s` and check `localhost:3000` is up and running

## Available Tasks Commands
- `rake csv:task1`
- `rake csv:task2`
- `rake csv:task3`
- `rake csv:task4`
- `rake csv:task5`
